# React Native Arabic Academy

أول أكاديمية عربية متخصصة في تعليم React Native خطوة بخطوة لبناء تطبيقات موبايل حقيقية بطريقة ممتعة وسهلة للمبتدئين والمتوسطين.

## عن المشروع

هذا المستودع يحتوي على أمثلة عملية، تمارين، ومشاريع صغيرة لتعليم تطوير تطبيقات موبايل باستخدام React Native من الصفر وحتى مرحلة نشر التطبيق على المتاجر.
الهدف هو توفير مرجع تعليمي مفتوح المصدر لكل من يرغب في تعلم React Native بالعربي مع أمثلة وتطبيقات واقعية.

## المتطلبات

- Node.js (إصدار حديث مستقر).
- npm أو yarn لإدارة الحزم.
- بيئة تطوير React Native:
  - إما باستخدام Expo CLI.
  - أو باستخدام React Native CLI مع Android Studio / Xcode.
